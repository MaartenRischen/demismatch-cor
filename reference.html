The reference library

Everything the framework rests on, in the order it argues itself.

Modern human suffering is, in many cases, accurate biological signaling in a mismatched environment - not disorder inside the individual. This library holds the supporting work behind that claim. It reads as the framework's own argument: what the organism needs, how those systems operate, where the environment creates a mismatch, how the inputs can be repaired, and the evidence the whole account is built on.
