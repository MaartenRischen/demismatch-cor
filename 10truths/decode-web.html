Decode Web

The read in place. Chrome extension. Any page you open, each flagged element tagged with the mechanism firing and the real input it is imitating.

Decode Web is the browser surface of the Demismatch Tools suite. It sits as a side panel in Chrome. You browse the web the way you already browse it - news feeds, social streams, product listings, articles - and it annotates what is on the page in place: which mechanism is firing, which evolved need the page is imitating, and why this page cannot resolve what it is activating.

Every annotation traces back to a named mechanism in the atlas. Decode Web does not invent a reading. It does not tell you how to feel about the page. It names the circuit, names the proxy, names the real input, and hands the reading back to you.

It runs on your own OpenRouter API key, pasted once into the side panel on first open. BYOK. Your traffic, your keys, your rate limit. No subscription, no server in the middle, no account.

Install · v0.4.2 · unpacked extension

Load it in Chrome

Decode Web is not yet published to the Chrome Web Store. It ships as an unpacked developer extension - a folder you load directly. Two minutes.

  1. Download decode-web.zip (via the button above) and unzip it anywhere on your machine. You will end up with a folder named decode-web.
  2. In Chrome, open chrome://extensions and toggle Developer mode on (top-right of that page).
  3. Click Load unpacked and select the unzipped decode-web folder.
  4. Open any page. Click the Decode Web icon in the Chrome toolbar to open the side panel.
  5. First run only: paste your OpenRouter API key into the side panel. The key is stored locally in the extension and never sent to a Demismatch server.
Sibling surface

Decode Web and Decode Talk are the same read on different surfaces. Decode Talk takes a situation you type; Decode Web takes what is already on your screen. Same spec underneath - the atlas at cor.demismatch.com, v4 mechanisms, cascade rules, WYATT.

If you want to see what the read sounds like in dialogue form first, try Decode Talk live. If you want the annotations on the page you are already reading, install Decode Web.
